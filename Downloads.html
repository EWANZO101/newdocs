SnailyCAD v4 Setup Guide

Complete deployment workflow

Prerequisites

Ensure Node.js 18+, PostgreSQL/MySQL database, and Git are installed before proceeding.

1

Initial Setup & Clone

Clone SnailyCAD v4 from GitHub and install dependencies. Ensure you have the latest stable release.

git clone https://github.com/SnailyCAD/snaily-cadv4.git
2

Configure Environment Variables

Create and customize your .env file with these essential settings:

DATABASE_URL - PostgreSQL/MySQL connection string
JWT_SECRET - Secure random string for authentication
PORT_API - API server port (default: 8080)
PORT_CLIENT - Client port (default: 3000)
3

Database Configuration

Set up your database connection and configure CAD-specific settings like department names, ranks, and locale.

üí° Supports PostgreSQL
4

Apply Updates & Migration

Run update.bat to install dependencies, run database migrations, and build the application.

5

Launch Your CAD

Use start.bat to start both API and client servers simultaneously.
