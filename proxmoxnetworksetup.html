Proxmox Configuration Example

ChatGPT can help you out here, but you will need to adapt it for your network setup. Alternatively, you can wait until the next update when the Proxmox service, as well as all VM configuration options, will be available.

Copy the configuration below:

    auto lo
    iface lo inet loopback
    
    auto enp34s0
    iface enp34s0 inet manual
    
    auto vmbr0
    iface vmbr0 inet static
        address 90.247.138.111
        netmask 255.255.255.0
        gateway 90.247.138.1
        bridge-ports enp34s0
        bridge-stp off
        bridge-fd 0
    
    # Copy the configuration for a new vmbr1 with a different IP address

Proxmox Network default configurations

        sudo nano /etc/network/interfaces
        iface enp34s0 inet manual
            
        auto vmbr0
        iface vmbr0 inet static
            address 192.168.0.14/24
            gateway 192.168.0.1
            bridge-ports enp34s0
            bridge-stp off
            bridge-fd 0
        
        sudo systemctl restart networking
